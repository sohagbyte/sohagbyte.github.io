আমরা তো আসলে প্রোগ্রামিং করি বা কাজ করি ডেটা নিয়ে। ডেটা ইজ এভ্রিথিং! 
জাভাস্ক্রিপ্টে Data Type জিনিসটা ডিফাইন করে দেয়, আমাদের ভ্যারিয়েবলে স্টোর করা ডেটার টাইপ কোনটা বা কোন ধরনের ডেটা আমরা স্টোর করছি, সেই জিনিসটা।

জাভাস্ক্রিপ্ট ভ্যারিয়েবল যেকোনো টাইপ এর ডেটা স্টোর করতে পারে। 
 **EcmaScript** এর লেটেস্ট স্ট্যান্ডার্ড অনুযায়ী জাভাস্ক্রিপ্টে প্রধানত ২ ধরনের ডেটা টাইপ আছে। এইগুলা হচ্ছে - 
 
 1. Primitive Data Type.
 2. Non-Primitive/Reference Data Type.

আবার Primitive Data Type আছে ৭ ধরনের এবং Non-Primitive/Reference Data Type আছে ১ ধরনের। 
## Primitive Data Type:

 Primitive Data Type গুলো জাভাস্ক্রিপ্টে Predefined থাকে বা আগে থেকেই define করা থাকে। এইটাকে Built-in Data Type ও বলা হয়। 
 ৭ ধরনের Primitive Type এর Data গুলো হচ্ছে-
 
 1. Number
 2. String
 3. Boolean
 4. Null
 5. Undefined
 6. Symbol
 7. BigInt
<br>
## Non-Primitive Data Type

জাভাস্ক্রিপ্টে Non-Primitive / Reference ডেটা টাইপ হচ্ছে ১ টা এবং যেইটা হচ্ছে --

1. Object

এছাড়াও Array, Function ও হচ্ছে Non-Primitive Type এর ডেটা। 
জাভাস্ক্রিপ্টে আসলে সবকিছুই Object এবং সেই হিসাবে Array, Function এইগুলাও Object. 
তাই এইগুলাকেও Non-Primitive Data Type বলা হয়।
